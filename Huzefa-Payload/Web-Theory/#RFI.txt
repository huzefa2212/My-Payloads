Remote File Inclusion

~ allows the attacker to upload a custom coded/malicious file on a website or server
~ the attacker uses a unvalidated file load to then load files from the hackers site to the the victim site
~ if a server pull documents then displays them as web content, test for RFI here eg http://victimsite.com/index.php?page=home , here we can see that it is pulling a document named home and displaying it as web content
~ look for "PHP include function" , as this is what is used to pull the webpages
