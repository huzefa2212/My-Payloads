Cross Site Tracing

TRACE
~ used for debug and connection analysis
~ easy to way to tell what an http client is sending and what the server is receiving
~ Returns contents of client HTTP requests
~ can be used to bypass xss protection header such as Httponly flag
~ can be used to steal cookies and credentials


https://www.smart-weblications.de/js/jsxc/lib/jsxc.dep.js


curl -v -X TRACE -H "Header: value=Testing" http://www.smart-weblications.de/
