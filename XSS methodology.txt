+-----------------Deliver Reflected XSS ------------------+

i) If a web application has banner advertisements, try to buy a banner and then create a page that steals the session token as soon as the user clicks on the banner advertisement.

ii) Create you own website that contains a script which will trigger a request to the victim web application resulting in an xss(stealing the session of victim web application of that user)
An Attacker may  use  search  engine  manipulation techniques to generate visits from suitable users, such as by placing relevant keywords within the site content and linking to the site using relevant expressions

iii) If a web application has “tell a friend” or send feedback to site administrators function which generates an e-mail with arbitrary content and recipients , the xss can be crafted in it.
